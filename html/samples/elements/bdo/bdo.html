When displaying text in a different direction than the surrounding content, use the bdo element. For example, to show Arabic text in right-to-left direction explicitly: مرحبا بالعالم (Hello, world).

If you want to override text direction to left-to-right, even within right-to-left content, you can write: 12345 ABC.

This is useful for ensuring correct display of mixed-direction content such as embedding English numbers inside Arabic text or correcting display issues in user interfaces.
