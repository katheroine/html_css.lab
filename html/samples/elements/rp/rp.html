In Japanese, the word 東京(とうきょう) means Tokyo. The rp element provides fallback parentheses for browsers that don’t support ruby, so you might see it written as 東京(とうきょう).

Similarly, for the phrase 学校(がっこう) (school), the parentheses appear only if ruby is not rendered properly.

This helps maintain readability in browsers that lack ruby support by enclosing the pronunciation or annotation in parentheses.
